# Installation
You can install TRL either from PyPI or from source:

## PyPI

Install the library with pip:

```bash
pip install trl
```

If you want to train models with ðŸ¤— PEFT, ðŸ¤— Diffusers, or DeepSpeed, install the additional dependencies as follows:

```shell
# Install PEFT dependencies
pip install trl[peft]

# Install diffusers dependencies
pip install trl[diffusers]

# Install DeepSpeed dependencies
pip install trl[deepspeed]
```

## Source

You can also install the latest version from source. First clone the repo and then run the installation with `pip`:

```bash
git clone https://github.com/huggingface/trl.git
cd trl/
pip install -e .
```

If you want the development install you can replace the pip install with the following:

```bash
pip install -e ".[dev]"
```
